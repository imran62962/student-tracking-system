Error Page

Bad Request (400) - The request was malformed or invalid.

Unauthorized (401) - Authentication is required and has failed or has not been provided.

Forbidden (403) - Access to the requested resource is forbidden.

Not Found (404) - The requested resource could not be found on the server.

Internal Server Error (500) - An unexpected condition was encountered by the server.

Service Unavailable (503) - The server is not ready to handle the request.

Unknown Error - An unknown error occurred.

Generic Error - An error occurred.
